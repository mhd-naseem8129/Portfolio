🩺 IoT Patient Monitoring System

A smart healthcare system that monitors patient vitals in real-time using IoT sensors and web technologies.

Dashboard

🛠️ Tech Stack

Python Flask IoT Sensors Scikit-learn HTML CSS JavaScript

❗ Problem

In many healthcare scenarios, continuous monitoring of patient vitals such as heart rate, temperature, and oxygen levels is difficult without constant supervision. This can delay critical medical decisions and affect patient safety.

✅ Solution

Developed an IoT-based patient monitoring system that collects real-time data from sensors and sends it to a web application. The system processes and stores the data, enabling doctors or users to monitor patient health remotely and take timely action.

🚀 Key Features

  • 📡 Real-time health data monitoring
  • ❤️ Tracks heart rate, temperature, and SpO₂
  • 🧠 Machine learning-based predictions
  • 🌐 Web dashboard for live data display
  • ⚡ Fast and efficient data processing

📈 What I Learned

Through this project, I gained hands-on experience with IoT systems, real-time data handling, and backend development using Flask. I also learned how to integrate machine learning models into web applications and design systems that solve real-world healthcare problems.
