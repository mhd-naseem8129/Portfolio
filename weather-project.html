🌦️ Weather Mood App

A smart web application that suggests activities based on real-time weather conditions.

🛠️ Tech Stack

Python Flask API Integration HTML CSS JavaScript

❗ Problem

Many people struggle to decide what activities to do based on changing weather conditions. Without proper guidance, this often leads to unproductive or poorly planned days.

✅ Solution

I developed a weather-based application that fetches real-time weather data using APIs and provides personalized activity suggestions based on the current weather and user mood. This helps users make smarter daily decisions and stay productive.

🚀 Key Features

  • 🌡️ Real-time weather data
  • 🎯 Mood-based activity suggestions
  • 📱 Simple and responsive UI
  • ⚡ Fast API integration

📈 What I Learned

Through this project, I improved my understanding of API integration, backend development using Flask, and building user-focused applications. I also enhanced my ability to design solutions that solve real-world problems.
