🚌 Bus Tracking System

A real-time web application to track college buses and improve student convenience.

🛠️ Tech Stack

Python Flask JavaScript HTML CSS WebSockets

❗ Problem

Students often face uncertainty regarding bus arrival times, leading to delays, missed buses, and inefficient time management. There is no easy way to track buses in real-time.

✅ Solution

Developed a real-time bus tracking system that allows students to monitor the live location of buses. The system helps improve planning, reduces waiting time, and enhances overall transportation efficiency.

🚀 Key Features

  • 📍 Real-time location tracking
  • 👨‍🎓 Student, Driver & Admin login system
  • ⚡ Live updates using web technologies
  • 📱 Responsive and user-friendly interface

📈 What I Learned

This project helped me understand real-time data handling, backend development using Flask, and designing systems with multiple user roles. I also gained experience in solving real-world problems through technology.
